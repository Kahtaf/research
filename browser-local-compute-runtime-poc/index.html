How it works

          

            Cloudflare hosts only the static app. The GCP relay routes opaque
            TCP streams by SNI. TLS, HTTP, API logic, MCP tools, and IndexedDB
            access all run in the active browser tab.
          

        

        
Cloudflare static host
  serves JS/CSS only
          |
          v
Browser tab
  UI textarea -> IndexedDB
  Browser Worker -> API + MCP handlers
  Browser TLS server terminates HTTPS
          ^
          | opaque TCP/TLS bytes
          |
GCP VM blind relay
  reads ClientHello SNI only
  routes <sessionId>.34.16.49.200.sslip.io
          ^
          |
